# 脑图

## 概述

### 什么是前后端
* CURD
  * Create/Update/Read/Delete

* 前端
  * layout/form
  * (span/div)/form
  * (block/inline)/action
  * 展示数据/收集数据
  * 使用 JavaScript 通过 浏览器提供的(ajax)接口与事件 和用户做交互

* API (application program interface)
  * 前后端约定的规范
  *  (http/https/rpc) => tcp/udp => server

* 后端
  *  存储数据/归纳数据
  *  编程语言(php/c/c++/nodejs) 与 数据库 做交互
